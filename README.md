# 🍴 Restoran Değerlendirme Uygulaması

Kullanıcıların ziyaret ettikleri restoranları puanlayabildiği, yorum yapabildiği ve yeni lezzet durakları keşfedebildiği bir mobil uygulamadır.

### ✨ Temel Fonksiyonlar
* **Kullanıcı Kayıt & Giriş:** Güvenli oturum yönetimi.
* **Restoran Listeleme:** Kategorilere ve puanlara göre filtreleme.
* **Puanlama Sistemi:** Kullanıcıların deneyimlerini 5 yıldız üzerinden değerlendirmesi.

### 🛠 Teknik Detaylar
* **Frontend:** Android Studio
* **Mimari:** Temiz kod prensiplerine uygun klasör yapısı.
* **Veri Yönetimi:** SQLite
